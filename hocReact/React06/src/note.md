# Redux

- Store --> Kho chứa các state global
- Reducer --> Hàm cập nhật State
- Action --> Object hành động được gửi từ Component lên Reducer
- Dispatch --> hàm gửi action lên Reducer
- Subscribe --> Hàm lắng nghe sự thay đổi của state
