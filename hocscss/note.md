# Ngôn ngữ tiền xử lý Css

- Ngôn ngữ lập trình
- tái sử dụng dễ dàng hơn
- quản lý code dễ hơn
- không bị nhàm chán

=> biên dịch sang css
=> ngôn ngữ tiền xử lý thường gặp

## Biên dịch tất cả folder

sass tenfolderscss:tenfoldercss

sass tenfolderscss:tenfoldercss --watch

sass duong-dan-file-scss

duong-dan-file-css

sass duong-dan-file-scss
